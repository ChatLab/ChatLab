ChatLab에 오신 것을 환영합니다. ChatLab은 **오픈소스 기반의 로컬 우선 대화 기록 분석 도구**입니다.

우리는 자신의 소셜 데이터를 스스로 이해할 권리가 있다고 믿습니다. 하지만 대화 기록은 나 혼자만의 것이 아니라 친구, 가족, 동료의 말과 신뢰도 함께 담고 있습니다. 이 도구를 사용하기 전에 아래 약관을 꼼꼼히 읽어주세요.

## 1. 핵심 기능 및 독립성

- **분석 전용, 내보내기 기능 없음**: 이 도구는 이미 내보내기된 대화 기록을 분석하는 용도로만 사용됩니다. 복호화, 패킷 캡처, 내보내기 도구는 제공하지 않습니다. 타사 대화 플랫폼에서 데이터를 얻는 행위는 전적으로 사용자 본인의 행동이며 책임입니다.
- **독립적인 타사 도구**: 본 프로젝트는 독립적인 오픈소스 프로젝트이며, **어떠한 대화 플랫폼, 기업, 단체와도 관련이 없습니다**.

## 2. 데이터 프라이버시 및 보안

- **순수 로컬 분석**: 기본적으로 대화 기록은 사용자의 기기에서만 저장 및 분석되며 **어디에도 업로드되지 않습니다**.
- **AI 기능 예외**: AI 기능을 활성화하기로 선택한 경우, 관련 대화 데이터가 사용자가 직접 설정한 타사 모델 제공자(OpenAI, DeepSeek 등)로 전송됩니다. 이 기능을 사용하기 전에 데이터의 민감도를 스스로 판단하고, **매우 사적인 내용은 전송하지 않도록** 주의하세요. 해당 타사 제공자의 개인정보 처리 방침은 본 프로젝트와 무관하므로, 반드시 직접 확인하시기 바랍니다.
- **기능상의 네트워크 연결**: 앱은 업데이트 알림, 사용 안내, 설정 데이터를 가져오기 위해 인터넷에 연결됩니다. 이 과정에서 로컬 데이터가 업로드되지는 않습니다.
- **익명 사용 통계**: 제품 개선을 위해 앱은 버전 번호, 운영체제 종류 등 **민감하지 않은** 데이터를 수집할 수 있습니다. 이 데이터에는 개인 정보가 포함되지 않으며, 설정에서 언제든지 이 기능을 끌 수 있습니다.

## 3. 데이터 이용 권한 및 사용 제한

대화 기록에는 나의 데이터뿐 아니라 대화 상대방이 남긴 말도 함께 담겨 있습니다. 이 도구를 사용할 때는 다음 원칙을 반드시 지켜주세요:

- **정당한 권한 원칙**: 본인이 **직접 참여한** 대화 기록만 처리할 수 있습니다. 해당 대화에 참여한 다른 사람들에게도 프라이버시 권리가 있습니다. 분석 대상에 식별 가능한 개인, 특히 **개인 대화 상대나 그룹 멤버**의 프라이버시가 포함된다면, 반드시 해당 인물의 명확한 동의를 받아야 합니다.
- **금지 행위**:
  - 타인의 개인정보를 무단으로 훔치거나, 감시하거나, 분석하는 행위를 **엄격히 금지합니다**.
  - 무단 침입, 계정 탈취, 데이터 가로채기 등 불법적인 방법으로 얻은 대화 기록을 분석하는 행위를 **엄격히 금지합니다**.
  - 분석 결과를 괴롭힘, 사기, 신상 공개(독싱) 등 타인의 권리를 침해하는 용도로 사용하는 행위를 **엄격히 금지합니다**.
  - AI가 생성한 내용을 포함하여, 본 소프트웨어가 생성한 분석 콘텐츠를 이용해 허위 또는 오해의 소지가 있는 정보를 조작하거나 유포하는 행위를 **엄격히 금지합니다**.

## 4. 위험 경고

- **공식 배포 채널**: 이 소프트웨어는 반드시 [chatlab.fun](https://chatlab.fun) 또는 [GitHub Releases](https://github.com/ChatLab/ChatLab/releases)에서만 다운로드하세요.
- **공급망 위험**: 본 프로젝트는 완전한 오픈소스이므로 누구나 재패키징할 수 있습니다. 타인이 배포하는 비공식 버전에는 API 토큰, 대화 기록, 로컬 데이터를 유출시키는 악성 코드가 포함될 수 있으니 항상 주의하세요.
- **결과의 정확성**: 소프트웨어나 AI가 생성한 분석 결과에는 오류나 환각(hallucination)이 포함될 수 있습니다. 이는 참고용일 뿐이며, 법적 증거나 공식적인 의사결정의 근거로 사용해서는 안 됩니다.

## 5. 면책 조항

- **소프트웨어의 목적**: 본 소프트웨어는 기술 연구, 학습, 교류 목적으로만 제공됩니다.
- **플랫폼 이용약관 준수**: 대화 기록을 얻고 분석하는 과정이 원본 플랫폼의 이용약관을 준수하는지는 전적으로 사용자 본인의 책임입니다. 사용자의 행위가 해당 약관을 위반하여 계정이 제한되더라도, 개발자는 어떠한 책임도 지지 않습니다.
- **프라이버시 책임**: 이 도구의 사용 방식이 타인의 프라이버시 또는 기타 정당한 권리를 침해하는 경우, 그 결과는 전적으로 사용자 본인의 책임입니다.
- **단독 책임**: 데이터 손실, 프라이버시 분쟁, 법적 책임을 포함하되 이에 국한되지 않는, 본 소프트웨어 사용으로 발생하는 모든 결과는 전적으로 사용자 본인의 책임입니다.
- **준거법**: 본 약관은 사용자가 속한 관할권의 관련 법률을 따릅니다. 본 약관의 일부가 현지의 강행법규와 충돌하는 경우, 현지 법률이 우선합니다.
- **약관 동의**: 본 소프트웨어를 다운로드, 설치 또는 사용함으로써 사용자는 본 약관의 모든 내용을 읽고, 이해하고, 동의했음을 인정합니다. 동의하지 않는 경우 즉시 사용을 중단하고 관련 파일을 모두 삭제해 주세요.
